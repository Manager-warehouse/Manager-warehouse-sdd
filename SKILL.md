# Diagram Toolkit

The canonical, compact skill package is:

[`.agents/skills/diagram-toolkit/SKILL.md`](.agents/skills/diagram-toolkit/SKILL.md)

It combines the former root diagram guidance with the usable parts of `diagram-skills-package`:

- one short workflow and diagram-selection table;
- validation rules for ERD, class, sequence, state, activity, and context diagrams;
- engine choice only when Mermaid is insufficient;
- Mermaid templates and a verification script.

Start at the canonical skill. The former `diagram-skills-package` folder now has a short [README](diagram-skills-package/README.md) and retains only advanced reference material/engines.
